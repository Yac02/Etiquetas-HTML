CONCEPTOS Y EJEMPLOS S2

ETIQUETAS

E. de Apertura

-La etiqueta de apertura comienzan con un corchetes angular "<"

EJEMPLO: "< nombre >"

E. de Cierre

-La etiqueta de cierre es similar a la apertura, pero incluye una barra diagonal antes del elemento

EJEMPLO: "< /nombre >"

E. Self Closing

-No requieren una etiqueta de cierre, porque se cierran ellas mismas

EJEMPLO: "< i m g >"

E. Bloque e inline

-Las etiquetas de bloque ocupan todo el espacio horizontal, mientras que las etiquetas de inline, solo ocupan el espacio de texto necesario

EJEMPLO: "< h1" vs "< span"

ATRIBUTOS

-Son valores adicionales que proporcionan información adicional sobre cómo debe comportarse un elemento

EJEMPLO: "href" (Para adicionar una URL)

ENCABEZADOS

Dan jerarquia tipografica

EJEMPLO: "< h1 >, < h2 >, < h3 >"
